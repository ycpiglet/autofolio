# EVIDENCE-2026-06-03-001 Requirement Elicitation

Status: imported pointer

This note anchors the Requirements Interviewer reference to requirement
elicitation practice. The actionable local protocol is maintained in
`agents/requirements_interviewer/SKILL.md` and `scripts/ambiguity_scan.py`.

## Local Rule

- Clarify substantive ambiguous requests before planning.
- Prefer a small batch of high-information questions.
- Stop once the request is decision-ready, not when every detail is exhausted.

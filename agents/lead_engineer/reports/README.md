# Reports Archive

Canonical archive for BRIEF and PLAN reports.

Required frontmatter follows `scripts/check_agent_docs.py`:

```yaml
---
type: report
id: BRIEF-YYYY-MM-DD-NNN
kind: BRIEF
date: YYYY-MM-DD
recorded_at: YYYY-MM-DDTHH:MM:SS+09:00
audience: Owner
scale: standard
title: Short title
author: Lead Engineer
insights_count: 1
decisions_count: 0
---
```
